```python
import time

import requests
```

Add `class RefeeEnergyProvider(EnergyProvider)` after `StakingEnergyProvider`.
It must:

1. Accept `tron_client=None` in `__init__` and store it.
2. In `acquire`, read `settings = config.REFEE` inside the method. If settings is
   `None`, log warning and return `False`.
3. Compute:
   ```python
   energy_required = minimum_energy_required or energy_to_provision
   amount = int((Decimal(energy_required) * settings.energy_overprovision_factor).to_integral_value(rounding=ROUND_CEILING))
   ```
4. POST to `f"{settings.api_base_url.rstrip('/')}/api/rent_resource/orders"` with:
   ```python
   headers={"X-API-Key": settings.api_key.get_secret_value()}
   json={
       "address": receiver,
       "amount": amount,
       "resource": "energy",
       "duration_label": settings.rent_duration_label,
   }
   timeout=10
   ```
5. Require HTTP `202`; on any other status log status code and response text, then
   return `False`.
6. Parse JSON, read `order_id = data.get("id")`, and return `False` if missing.
7. Poll `GET {base}/api/rent_resource/orders/{order_id}` every
   `settings.poll_interval_sec` until `settings.timeout_sec` expires.
8. Treat `status == "delegated"` or `status == "completed"` as success for polling.
9. Treat `failed`, `insufficient_funds`, and `canceled` as terminal failures.
10. After delegated/completed, call
    `(self.tron_client or ConnectionManager.client()).get_account_resource(receiver)`
    and require `EnergyLimit >= energy_required`.
11. Catch `requests.RequestException`, `ValueError`, and `KeyError`; log with
    `logger.exception` or `logger.warning` and return `False`.
12. Implement `release(self, receiver)` as a no-op with an INFO log that re:Fee
    returns energy after rent expiration.

Update `get_energy_provider(tron_client=None)`:

```python
def get_energy_provider(tron_client=None) -> EnergyProvider:
    if config.ENERGY_SOURCE == "refee":
        return RefeeEnergyProvider(tron_client=tron_client)
    return StakingEnergyProvider(tron_client=tron_client)
```

Do not change `StakingEnergyProvider` behavior.
